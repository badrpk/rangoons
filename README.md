# Rangoons 🛒

Open **WhatsApp-integrated e-commerce** monorepo.

## Layout

| Path | What |
|------|------|
| `apps/core` | Main store + WhatsApp (from rangoons-core) |
| `apps/live` | Live/flash commerce (from rangoons-live) |
| root modules | Ecosystem bridges (AI, Cast, Lang, …) |

## Download & use

```bash
git clone https://github.com/badrpk/rangoons.git
cd rangoons/apps/core
npm install   # if package.json present
# follow apps/core/README.md
```
